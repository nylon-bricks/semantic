Edit Metadata

Key files are located under /src/constants:

  • menu.ts: Menu items for the sidebar and header

  • metadata.ts: Site and author information

  • profile.ts: Profile card details on the main page

These files control how your site looks and feels ‚Äî make them your own.

Create a Post

To write a new blog post, add a folder (your post's slug) inside /content/posts, then create an .mdx file with the required frontmatter.

The following frontmatter fields are available for posts:

(Fields marked with * are required)


  • title*: Main title of the post

  • subtitle*: Short description or subtitle

  • createdAt*: Original publication date

  • modifiedAt*: Last updated date

  • coverImage*: Top cover image (supports relative or absolute paths)

  • category*: Single category assigned to the post

  • tags: One or more keywords for filtering

  • comments: Enable comments via Giscus (default: false)

To update the /about page, simply modify the /content/pages/about.mdx file.
